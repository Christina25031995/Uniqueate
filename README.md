# Uniqueate — лендинг

Статический сайт. Ничего собирать не нужно.

## Файлы
- `index.html` — страница (открывается прямо в браузере)
- `support.js`, `image-slot.js` — рантайм (нужны рядом с index.html)
- `images/` — изображения

## Залить на GitHub Pages
1. Создайте репозиторий и загрузите ВСЕ файлы из этой папки, сохранив структуру (`index.html` в корне, папка `images/` рядом).
2. Settings → Pages → Source: `main` / `/ (root)` → Save.
3. Через минуту сайт будет доступен по адресу `https://<ваш-логин>.github.io/<репозиторий>/`.

Важно: держите `support.js`, `image-slot.js` и папку `images/` в той же папке, что и `index.html`.
